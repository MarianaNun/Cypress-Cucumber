# cypress-cucumber-example
Initial example of using Cypress with Cucumber.

# Run tests
# Run in Cypress UI
```
npx cypress open
```  

# Run in terminal
```
npx cypress run
npm cy:uitest
npm cy:apitest

```UI
npm run cy:uitest
```

```API
npm run cy:apitest
```

If you want to use Chrome browser during non-interactive run, pass its name

```Chrome UI 
npm run cy:uitest --browser chrome 
```
